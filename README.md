# ConfigMod

幕オン2のUIの部分的非表示・キーコンフィグ拡張・便利機能MOD<br>

- 起動: `F1`キーで設定ウィンドウを開閉
- UIの部分的な非表示（アクションボタンなど）
- 遺跡で使用するアイテム（Key1, Key2, Key3）をキーボードやコントローラーの任意のボタンに割り当て
- メニュー画面の開閉をキーに割り当て
- FPS表示
- スクリーンショット機能
- ゲームウィンドウが非アクティブのときに音量をミュートするオプション

---

## 必要環境

- Windows 版の幕末志士の幕末オンライン2 のゲームファイル
- BepInEx 導入済みの環境
- [OfflinePatch](https://github.com/Bakuon2-Modding/OfflinePatch) 導入済みの環境

> MOD環境のセットアップ方法は [Bakuon2-Modding/SetupGuide](https://github.com/Bakuon2-Modding/SetupGuide) を参照してください。

> オフラインパッチのセットアップ方法は [Bakuon2-Modding/OfflinePatch](https://github.com/Bakuon2-Modding/OfflinePatch) を参照してください。

---

## インストール

[Releases](../../releases) から最新の `ConfigMod.zip` をダウンロードして解凍し、`ConfigMod.dll` をゲームフォルダの `BepInEx/plugins/` に配置してください。

```
bakuon2/
  BepInEx/
    plugins/
      BakuonOfflinePatch.dll   ← OfflinePatch（必須）
      ConfigMod.dll          ← ここに置く
```

## MOD UI の開き方

**F1** キーで設定ウィンドウを開閉します。

ウィンドウは3つのタブで構成されています：

| タブ | 内容 |
|---|---|
| キーコンフィグ | 各アクションのキー割り当て変更 |
| 設定 | FPS表示・非アクティブ時ミュート |
| UI表示設定 | ゲームUIの各パーツの表示/非表示 |

---

## 主な機能

### キーコンフィグ

キーボード・XInputコントローラーのボタンを各アクションに割り当てられます。<br>
（XBOXコントローラー以外で動作確認をしていないため、他のコントローラーでは動作しない可能性があります）

| アクション | デフォルト | 説明 |
|---|---|---|
| スクリーンショット | 未割り当て | `Screenshots/` フォルダに PNG 保存 |
| UI 表示/非表示 | 未割り当て | ゲーム UI 表示/非表示のトグル |
| メニュー 開閉 | 未割り当て | メニュー画面を開閉（ダンジョン内対応） |
| アイテム1使用 | 1 | ステアゲ アイテムスロット1 |
| アイテム2使用 | 2 | ステアゲ アイテムスロット2 |
| アイテム3使用 | 3 | ステアゲ アイテムスロット3 |

### 設定

- **FPS 表示** — 画面右上に現在の FPS をオーバーレイ表示
- **非アクティブ時にミュート** — ウィンドウのフォーカスが外れたときに音量をミュート

### UI表示設定

ゲーム画面の UI パーツを個別にオン/オフできます。

- UIのオンオフボタン
- アクションボタン
- マイカードボタン
- ルーム一覧ボタン

## ビルド（開発者向け）

ビルドにはゲームの DLL ファイルが必要です。本リポジトリをゲームフォルダ内の `ConfigMod/` として配置してください。

```
bakuon2/
  ConfigMod/   ← このリポジトリをここにクローン
  bakuon2_Data/
  BepInEx/
  ...
```

```bash
cd ConfigMod
dotnet build
```

ビルドに成功すると `BepInEx/plugins/ConfigMod.dll` に自動コピーされます。
